CASino is an easy to use Single Sign On (SSO) web application written in Ruby.

Download View on GitHub

Various backends

CASino supports a wide range of different authentication sources such as an LDAP directory or an SQL database.

Wide support

CASino supports a wide range of client platforms (including Ruby, Java, PHP, .NET, Apache and many more).

Localization

CASino automatically detects the user's preferred language and presents the appropriate interface.
